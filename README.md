# Churn Analysis (Python)

This project analyzes customer churn using Python.

## Overview

The goal is to understand customer behavior and identify patterns that lead to churn.

## Dataset

* Telco Customer Churn dataset
* Includes customer demographics, services, and billing information

## Features

* Data cleaning and preprocessing
* Churn rate calculation
* Churn analysis by contract type
* Visualization using Matplotlib

## Results

* Majority of customers did not churn
* Customers with month-to-month contracts have higher churn rates

## Tools

* Python
* Pandas
* Matplotlib

## How to Run

```bash
cd data
python analysis.py
```
