# Churn Analysis (Python)

This project analyzes customer churn using Python (Pandas, Matplotlib).

## Dataset
- Telco Customer Churn dataset

## Features
- Data cleaning
- Churn rate calculation
- Churn analysis by contract type
- Data visualization

## Results
- Majority of customers did not churn
- Customers with month-to-month contracts have significantly higher churn rates

## Tools
- Python
- Pandas
- Matplotlib
